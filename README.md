# PocketTech (React + Node.js)

Includes:
- Devices catalog (phones, laptops, tablets, accessories, wearables)
- Local images packaged (SVG files served from Node server)
- Search + category filter
- Cart + multi-step checkout
- Sell device form with photo upload (Multer)
- Orders + contact + sell requests saved in server/data/*.json

## Run
From project root:

```bash
npm run install:all
npm run dev
```

Frontend: http://localhost:5173  
Backend:  http://localhost:5000
